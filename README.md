## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

![Customer Section](https://github.com/bhuvan2205/ecommerce/assets/56767330/2dad3115-3201-4322-9d84-3e7cb20e9c6d)

![Admin Section](https://github.com/bhuvan2205/ecommerce/assets/56767330/58654c8e-e69a-45ab-8e30-8a584bf52d65)


You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/basic-features/font-optimization) to automatically optimize and load Inter, a custom Google Font.
